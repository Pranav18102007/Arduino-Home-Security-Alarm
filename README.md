## Home Security Alarm (Arduino)
This project is an automatic security alarm. 


## Components:
-Arduino uno
-Buzzer
-PIR sensor
-Jumper wires + Breadboard
-4 LEDs


## How It Works
If an intruder enters after the project is turned on it sounds an alarm and warns the owner.
